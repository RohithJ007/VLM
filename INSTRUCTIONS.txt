Problem: "Camera not found"
Solution:
  - Check camera is connected
  - Close other apps using camera (Zoom, Teams, etc.)
  - Try different camera: Change CAMERA_INDEX in config.py
  - Run: python camera.py (test camera separately)

Problem: "Model not found"
Solution:
  - Run: python download_model.py
  - Check internet connection
  - Ensure 5GB+ free disk space

Problem: "Out of memory"
Solution:
  - Edit main.py: set use_quant = True (line ~440)
  - Close other applications
  - Restart computer

Problem: Slow inference
Solution:
  - First run is always slower (loading)
  - Subsequent runs are faster
  - Enable quantization for 8GB RAM systems

Problem: PowerShell script error
Solution:
  - Use: .\venv\Scripts\python.exe instead of activate
  - Or run commands directly with full python path
